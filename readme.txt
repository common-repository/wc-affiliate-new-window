This plugin allows the admin to select a support ribbon that will be displayed in the lower left corner of your WordPress site.

== Description ==

This plugin uses jQuery to open external/affiliate links in a new window.

==For more information visit==

[Woo-WP External Links](http://portalplanet.net/wordpress-themes-plugins/woo-wp-external-links/)

== Installation ==

1. Upload `wcaff-affiliate-new-window` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. That's it! Your affiliate/external links should open in a new window.
4. To open in the same window simply deactivate this plugin.
